    Frequently asked questions
    What is the Gemini watermark?

    Google Gemini adds a small semi-transparent 4-point star watermark to AI-generated images. It appears in the bottom-right corner. The watermark is 48×48 px on smaller images and 96×96 px on images larger than 1024×1024.

    Is this tool safe to use?

    Yes. This tool runs entirely in your browser using the Canvas API. Your images are never uploaded to any server — all processing happens locally. You can verify this in your browser's network tab.

    How does the removal work?

    The tool uses reverse alpha blending to mathematically recover original pixel values. The Gemini watermark is a semi-transparent white overlay with a known alpha pattern. We reverse: original = (watermarked − alpha × 255) / (1 − alpha) per channel.

    What formats are supported?

    PNG, JPEG, and WebP. Output is always lossless PNG to prevent compression artifacts.

    Can I process multiple images?

    Yes — drag and drop or select multiple files at once. Each is processed in sequence with individual download buttons.

    Does this strip EXIF metadata?

    Yes. Processing through the Canvas API automatically removes all EXIF data. The output PNG contains only pixel data.

    About

    A free, open-source tool that removes the visible star-shaped watermark Google Gemini adds to AI-generated images. Unlike AI inpainting tools, this mathematically reverses the exact blending operation — pixel-perfect, zero quality loss.
